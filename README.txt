Instructions: Assignment 1
==========================

Compilation and Execution:

1. To compile the program, use the following command:
	make TDRend
	
2. To run the program, use the following command:
	make run
	
User Interactions:

1. Translation:
	Hold and drag the mouse using right click to specify direction and
	distance to translate the object.
	
2. Addition and deletion of points:
	Addition: Click on the screen using left mouse button to add points.
	Deletion: Click on the existing point to remove it.

3. Scaling:
	Use '+' and '-' keys to zoom in or out the rendered curves.

4. Rotation:
	Use 'r' and 'l' keys to rotate the curves in clockwise and anti-
	clockwise about the centre of the curves.
